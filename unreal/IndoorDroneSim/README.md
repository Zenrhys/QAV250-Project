# Indoor Drone Simulator — Unreal Engine 5
Digital twin environment for the QAV250 indoor navigation project (Phase 8).

## Setup
1. Open this folder as an Unreal Engine 5.3 project (required for UAV Toolbox co-sim).
2. Install the **UAV Toolbox Interface for Unreal Engine Projects** support package.
3. See `simulink/plant_model/qav250_plant_unreal.slx` for the Simulink co-sim model.

> **Note**: UE 5.7 may be used for standalone environment development, but co-simulation
> with MATLAB R2025b UAV Toolbox requires UE 5.3. See implementation plan for details.
